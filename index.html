GitHub – Teljes Alap Útmutató

Előnyök, hátrányok, parancsok és fogalmak

✅ Előnyök

  • Verziókezelés (Git)
  • Könnyű csapatmunka
  • Ingyenes publikus projektek
  • GitHub Pages (weboldal hosting)
  • Nagy fejlesztői közösség

❌ Hátrányok

  • Kezdőknek nehéz lehet
  • Git parancsok bonyolultak
  • Internet kell a használathoz

💻 Alap Git parancsok

Repository létrehozása:

git init

Fájlok hozzáadása:

git add .

Commit (mentés):

git commit -m "első commit"

Kapcsolódás GitHubhoz:

git remote add origin URL

Feltöltés:

git push -u origin main

Frissítés letöltése:

git pull

📚 Fontos fogalmak

  • Repository (repo): Projekt tároló
  • Commit: Egy mentési állapot
  • Branch: Külön fejlesztési ág
  • Merge: Két ág egyesítése
  • Pull request: Kód módosítás javaslat
  • Clone: Repo letöltése gépre
